# Kuhluhs

An illustration of an async action creator.

## Setup

After cloning this repo:

```sh
npm install
```

To run the server:

```sh
npm run build
npm run server
```

Then go to [http://localhost:3000](http://localhost:3000)

Or, to run `webpack-dev-server`:

```sh
npm start
```

And go to [http://localhost:8080](http://localhost:8080)
